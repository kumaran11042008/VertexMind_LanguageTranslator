# AI Language Translation Tool

## Project Description
This project is an AI-powered Language Translation Tool developed using Python and Tkinter GUI.

The application allows users to:
- Enter text
- Select source language
- Select target language
- Translate text instantly

The project uses Google Translation API through the deep-translator library.

---

## Technologies Used
- Python
- Tkinter
- deep-translator

---

## Features
- Simple GUI interface
- Multiple language support
- Fast translation
- Error handling
- Beginner-friendly design

---

## How to Run

1. Install Python
2. Install required library:

pip install deep-translator

3. Run the project:

python translator.py
